Curriculum

Six Games. Six Skills.

Each game targets a specific skill. Together they form a progressive set of drills and challenges that build reaction speed, memory, focus and adaptability.

Game 01
⚡
Reaction Control
Eye-Hand Reflex
A cell lights up on the court. The student must point and trigger before it disappears. Speed and precision are everything. The controller adds a motor coordination layer that touchscreen cannot replicate.
Skill: Response time · Controller precision · Motor coordination
Game 02
🧩
Pattern Recognition
Sequential Memory
Numbered cells light up one by one across the 5×10 court. The student memorizes the positions, then the cells go dark and they tap them back in exact order. Difficulty scales from 5 cells to 20.
Skill: Short-term memory · Spatial recall · Sequential accuracy
Game 03
🔢
Flash Recall
Multi-Target Memory
Multiple cells flash simultaneously across 50 positions. The student has fractions of a second to register all lit cells, then must find and tap every one in any order. Demands full court awareness.
Skill: Peripheral awareness · Multi-target memory · Visual scanning
Game 04
🎵
Sequence Control
Working Memory
Cells light up one by one in gold with musical notes. The student watches the full sequence across the court, then replays it in the exact same order. Each correct tap plays its note. Wrong order reveals the pattern.
Skill: Working memory · Order retention · Motor replay
Game 05
🌱
Echo Recall
Change Detection
The original pattern flashes in gold. Then the echo pattern flashes in lime — identical except for one cell that has moved. The student must identify and tap the changed cell before the timer runs out.
Skill: Comparative memory · Change detection · Mental flexibility
Game 06
🎯
VR Number Hunt
Visual Target Search
50 numbers fill the VR court. A target number appears. The student scans the full court and triggers before the shot clock expires. The flagship experience — pure eye-hand reaction under time pressure.
Skill: Visual scanning · Target recognition · Speed under pressure
Learning Path

A Progressive Training Journey

Students advance through a structured four-stage path — from free practice to immersive VR competition.

1
Practice — Mobile & Chromebook
Students begin on the free 5×5 court on their phone or school Chromebook. Low pressure, self-paced. They learn the mechanics of each game and build baseline reaction skills at their own speed.
2
Class Competition — Promethean Board
The teacher launches the 5×10 competitive court on the Promethean board. The whole class plays simultaneously — students on Chromebooks, scores displayed live on the big screen. Class-wide competition drives engagement.
3
VR Immersion — Pico Headset
Students put on a Pico VR headset and enter the full 5×10 court in virtual reality. The controller adds a precision layer that elevates every game. The same skills — harder execution. The most immersive training experience available in K-12.
4
Tournament & Leaderboard
All scores post to the Reactific leaderboard in real time — daily, weekly and all-time rankings. Students compete across classes, campuses and programs. Teachers track progress. Winners are recognized. The data never lies.
