The Curriculum

It reads like one game.
It teaches like five chapters.

Each chapter challenges a different way of thinking. Played in order, they build on one another — from holding a memory, to reading a pattern, to reacting in open space.

1
Chapter One Change Detection
👁️ Echo Recall

A pattern flashes, then echoes back with one cell changed. The student holds the original in mind and spots the single difference — training the working memory that underlies nearly all learning.

2
Chapter Two Sequential Memory
🧩 Pattern Recognition

Numbered cells light in order, then vanish. The student replays the sequence from memory — building the ordered recall that reading, math, and following instructions all depend on.

3
Chapter Three Spatial Memory
⚡ Flash Recall

Several cells flash at once, then disappear. The student finds them all — strengthening the spatial memory used to navigate, organize, and picture things in the mind's eye.

4
Chapter Four Audio & Order Memory
🎵 Sequence Control

Cells light one by one, each with its own tone. The student replays the exact order — linking sound to sequence, the same pairing that underpins music, language, and rhythm.

5
Chapter Five Spatial Reflex
🎯 Reaction Control

The final chapter leaves the grid behind. Targets appear anywhere in open space and the student reacts on instinct — pure speed and spatial reflex, the course's fast-paced finale.
