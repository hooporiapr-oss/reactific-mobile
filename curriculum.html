1Curriculum Overview

The Ritnome Beats eSports Curriculum is a structured training program designed to develop automatic recall, memory control, quick decision-making, accurate execution, and sustained engagement.

The curriculum is built around four core drills:

  • REFLEX
  • REPLAY
  • RECALL
  • RITMO

Each drill develops a different part of memory performance. Together, they create a complete training system that challenges players to observe information, retain it, retrieve it, compare it, make decisions, and execute a correct response.

All four drills use the same Ritnome grid, the same interaction system, and one consistent tempo of 100 beats per minute.

This shared structure allows students to focus on the training itself instead of learning new controls or rules for every activity.

Four drills. One grid. One tempo. Continuous development.

Launch Status. REFLEX and REPLAY are available at launch as the curriculum’s reaction-centered drills. RECALL and RITMO will be introduced in a subsequent release as the curriculum’s memory-focused expansion.

2Curriculum Purpose

The purpose of the Ritnome curriculum is to help students strengthen their ability to:

  • Pay attention to visual information
  • Hold information in memory
  • Recall information in the correct order
  • Recognize multiple pieces of information quickly
  • Compare what they saw before with what they see now
  • Transfer remembered information into accurate action
  • Maintain performance under controlled time demands
  • Improve through repeated, measurable practice

Ritnome applies a principle commonly seen in sports training.

Athletes repeat movements until those movements can be performed automatically. Ritnome uses structured repetition to help students develop more automatic and dependable recall.

Muscle memory makes movement automatic. Ritnome makes recall automatic.

3Curriculum Philosophy

Ritnome is based on five instructional principles.

Active Participation

Students cannot progress by watching passively. Every round requires observation, memory, decision-making, and response.

Purposeful Repetition

Students repeat core memory processes, but the patterns and challenges change. This provides repetition without requiring students to perform the exact same task every time.

Consistent Structure

Every drill uses the same grid and 100 BPM tempo system. Students can move from one drill to another without spending valuable training time learning a new interface.

Measurable Progress

Performance is recorded through accuracy, successful rounds, levels, and scores. Students and educators can see improvement over time.

Progressive Challenge

The rhythm remains constant, but the memory demand increases. Ritnome automatically introduces more information, longer sequences, more complex patterns, and higher mastery expectations as the student improves.

4The Four Curriculum Strands

THE WARMUP: FOUNDATIONAL DRILL

Pure Reaction and Baseline Speed

Drill Description

Before any of the four curriculum strands, every Ritnome session opens with the Warmup. A single cell lights up at a time across the full grid, and the student taps it as quickly as possible. There is no sequence to hold, no pattern to compare, and no memory requirement of any kind — the Warmup measures reaction speed alone.

Primary Skill

Reaction speed and session readiness

Supporting Skills

  • Visual attention
  • Immediate response
  • Hand-eye coordination
  • Comfort with the 100 BPM tempo
  • Confidence before cognitive demand increases

Learning Objectives

  1. Respond to a visual target as quickly as possible.
  2. Sustain attention across a full round of the grid.
  3. Build comfort with the grid, the response method, and the 100 BPM tempo before a memory task is introduced.
  4. Establish a personal reaction-speed baseline at the start of every session.

A Confidence Builder and a Diagnostic Tool

The Warmup serves two purposes at once. First, because it requires no memory and cannot produce a discouraging result, it gives every student an early, achievable success before the more demanding memory strands begin. Second, because the Warmup measures raw response time and converts it into a beats-per-minute (BPM) reaction level, it gives the educator an immediate, individual reaction-speed indicator for each student, recorded at the start of every session.

Because the Warmup's reaction level is expressed in the same BPM unit as the training tempo itself, a student's personal reaction speed can be read and compared directly alongside the fixed 100 BPM standard used across all four strands, without requiring a separate testing instrument.

Evidence of Readiness

A student demonstrates readiness when the Warmup is completed for all fifty targets, with an individual reaction-speed level, expressed in BPM, recorded for that session.

Educational Connection

  • Identify a student’s current response speed before a task requiring memory
  • Build a personal reaction-speed profile over time
  • Compare reaction speed across sessions
  • Enter the memory strands already comfortable with the grid and the tempo

STRAND 1: REFLEX

Quick Decisions and Full-Grid Awareness

Drill Description

In REFLEX, several cells flash at the same time. The student must identify every active cell and tap all of them before the opportunity ends.

Unlike a simple one-target activity, REFLEX requires the student to observe a wider area, recognize several targets, make a quick decision, and respond without overlooking any part of the grid.

Primary Skill

Quick and accurate decision-making

Supporting Skills

  • Immediate visual recognition
  • Multiple-target awareness
  • Full-grid attention
  • Visual scanning
  • Response control
  • Accuracy under time demands

Learning Objectives

By participating in REFLEX, students will learn to:

  1. Observe information across the full Ritnome grid.
  2. Identify several active cells during one presentation.
  3. Distinguish active cells from inactive cells.
  4. Make quick decisions without random tapping.
  5. Respond to every required target.
  6. Balance speed with accuracy.
  7. Maintain awareness as the number of targets increases.

Progression

The level of challenge may increase through:

  • More cells flashing at once
  • Wider spacing between active cells
  • More complex visual arrangements
  • Greater demands on full-grid scanning
  • Higher accuracy requirements
  • More consistent performance required before advancement

The 100 BPM tempo remains unchanged. The difficulty increases because the student must recognize and process more information.

Evidence of Mastery

A student demonstrates mastery when the student can consistently identify and select all required cells without unnecessary selections or missed targets.

Educational Connection

The abilities trained in REFLEX may be applied when students must:

  • Locate important details quickly
  • Scan a page, chart, diagram, or problem
  • Identify several correct elements
  • Make decisions from visual information
  • Work accurately within a limited period
  • Remain attentive across a larger visual area

STRAND 2: REPLAY

Memory-to-Execution Transfer

Drill Description

In REPLAY, the student watches a complete sequence and then physically runs it back from memory.

After the sequence has ended, the student must execute the entire sequence correctly. The drill emphasizes not only remembering the information but also converting that remembered information into organized action.

REPLAY challenges the student to hold a complete sequence internally and then perform it with control.

Primary Skill

Memory plus execution

Supporting Skills

  • Whole-sequence retention
  • Memory-to-action transfer
  • Ordered execution
  • Pattern reproduction
  • Response consistency
  • Performance control

Learning Objectives

By participating in REPLAY, students will learn to:

  1. Watch a complete sequence without responding too early.
  2. Hold the full sequence in memory.
  3. Organize the remembered information before acting.
  4. Execute the sequence in the correct order.
  5. Complete the sequence without losing focus midway.
  6. Connect visual memory with physical response.
  7. Improve consistency across repeated attempts.

Progression

The level of challenge may increase through:

  • Longer complete sequences
  • More changes in direction across the grid
  • Greater distance between sequence locations
  • More complex movement patterns
  • Higher accuracy requirements
  • More uninterrupted execution required for mastery

Evidence of Mastery

A student demonstrates mastery when the student can retain and reproduce a complete sequence accurately, in order, and with controlled execution.

Educational Connection

The abilities trained in REPLAY may be applied when students must:

  • Remember and carry out a full set of instructions
  • Transfer learned information into action
  • Complete multistep procedures
  • Reproduce demonstrated processes
  • Organize actions before beginning a task
  • Perform a learned sequence independently

STRAND 3: RECALL (COMING SOON)

Memory and Sequential Retrieval

Drill Description

In RECALL, a sequence of cells lights up on the Ritnome grid. The student watches the sequence, holds the locations and order in memory, and then taps the cells back in the exact order presented.

The student must remember both:

  • Which cells appeared
  • The order in which they appeared

RECALL develops the ability to retrieve visual information in an organized sequence.

Primary Skill

Sequential memory

Supporting Skills

  • Visual location memory
  • Ordered retrieval
  • Concentration
  • Attention to sequence
  • Response accuracy
  • Automatic recall

Learning Objectives

By participating in RECALL, students will learn to:

  1. Observe a visual sequence from beginning to end.
  2. Retain the location of each presented cell.
  3. Remember the correct order of the sequence.
  4. Reproduce the sequence accurately.
  5. Maintain focus as sequences become longer.
  6. Recover from mistakes and improve during later attempts.
  7. Demonstrate increasingly consistent recall.

Progression

The level of challenge may increase through:

  • Longer sequences
  • More possible grid locations
  • Greater similarity between sequence patterns
  • Increased accuracy requirements
  • Multiple successful rounds required for mastery
  • Reduced dependence on repeated demonstrations

Evidence of Mastery

A student demonstrates mastery when the student can repeatedly reproduce sequences at the required level with accurate order and consistent performance.

Educational Connection

The abilities trained in RECALL may be applied when students must:

  • Remember steps in a process
  • Follow directions in order
  • Recall number or letter sequences
  • Retain the order of events
  • Complete multistep academic tasks
  • Remember procedures without repeated prompting

STRAND 4: RITMO (COMING SOON)

Memory, Comparison, and Change Detection

Drill Description

In RITMO, the student watches an original pattern and holds it in memory. The pattern is then presented again with a change.

The student must compare the second presentation with the remembered first presentation and identify exactly what changed.

RITMO requires the student to create a mental representation of the original pattern, maintain that information, compare two presentations, and make an accurate decision.

Primary Skill

Memory plus quick decisions

Supporting Skills

  • Pattern memory
  • Visual comparison
  • Change detection
  • Attention to detail
  • Mental holding
  • Accurate decision-making

Learning Objectives

By participating in RITMO, students will learn to:

  1. Observe and retain an original visual pattern.
  2. Hold that pattern in memory during a second presentation.
  3. Compare remembered and current information.
  4. Detect the specific change between two patterns.
  5. Avoid responding to information that remained the same.
  6. Make a quick and accurate comparison decision.
  7. Recognize increasingly subtle changes.

Progression

The level of challenge may increase through:

  • More cells included in the original pattern
  • More complex pattern arrangements
  • Smaller or less obvious changes
  • Changes appearing in different grid locations
  • Greater similarity between the first and second patterns
  • Higher accuracy requirements before advancement

Evidence of Mastery

A student demonstrates mastery when the student can consistently identify the exact change between the original and second presentation.

Educational Connection

The abilities trained in RITMO may be applied when students must:

  • Compare two pieces of information
  • Detect errors or differences
  • Review written work
  • Notice changes in numbers, symbols, diagrams, or text
  • Distinguish correct information from altered information
  • Pay close attention to important details

5Curriculum Learning Outcomes

After consistent participation in the Ritnome curriculum, students should demonstrate growth in the following areas:

Memory Performance

Students will improve their ability to hold, retrieve, and reproduce visual information.

Automatic Recall

Students will become more efficient at retrieving practiced information without unnecessary hesitation.

Sequential Control

Students will improve their ability to remember and execute information in the correct order.

Quick Decision-Making

Students will make faster decisions while maintaining accuracy.

Visual Attention

Students will improve their ability to observe individual targets, multiple targets, full patterns, and changes across the grid.

Memory-to-Action Transfer

Students will strengthen the connection between remembering information and performing a correct response.

Performance Consistency

Students will work toward repeating successful performance across several rounds rather than succeeding only once.

Self-Monitoring

Students will become more aware of their own accuracy, progress, strengths, and areas that require additional practice.

6Instructional Structure

A Ritnome session may include all four drills or focus on one specific curriculum strand.

A recommended session structure is:

A. Preparation

The educator introduces the purpose of the session and identifies the skill being trained.

Examples:

  • “Today we are training sequence memory.”
  • “Today we are working on detecting changes.”
  • “Today we are practicing quick decisions without losing accuracy.”

B. Familiarization

Students complete introductory rounds to become comfortable with the drill and grid.

C. Focused Training

Students complete structured rounds within the selected drill.

D. Progressive Challenge

The system increases the difficulty when the student demonstrates the required level of mastery.

E. Performance Review

Students review scores, accuracy, and completed levels.

F. Reflection

Students briefly identify what helped or interfered with their performance.

Possible reflection prompts include:

  • What helped you remember the sequence?
  • Did you respond too quickly or too slowly?
  • Which part of the grid did you overlook?
  • What will you do differently in the next round?

7Tempo Standard

All four drills operate at one standard tempo:

100 BPM

Students do not select beginner, intermediate, or advanced speed tiers.

The consistent tempo creates a standardized training environment across:

  • Drills
  • Students
  • Classrooms
  • Teams
  • Devices
  • School competitions

The tempo does not become faster as students advance.

Instead, Ritnome increases the cognitive and performance demand through:

  • More information
  • Longer sequences
  • More active cells
  • More complex patterns
  • More difficult comparisons
  • Higher accuracy expectations
  • Greater consistency requirements

The tempo remains constant. The challenge grows.

8Automatic Challenge Progression

Ritnome automatically increases the level of challenge as students demonstrate mastery.

Students are not required to choose their own difficulty. The system uses successful performance to determine when the student is ready for a more demanding level.

Depending on the drill, automatic progression may introduce:

DrillHow the Challenge Increases
REFLEXMore simultaneous targets and wider grid coverage
REPLAYLonger full sequences and more demanding execution patterns
RECALLLonger and more complex sequences
RITMOMore complex patterns and less obvious changes

This model helps prevent two common problems:

  • A task that remains too easy and becomes repetitive.
  • A task that becomes too difficult before the student is prepared.

Students progress by demonstrating performance, not simply by spending time in the program.

9Engagement System

Ritnome is designed to enforce active engagement.

Every round requires the student to perform one or more of the following actions:

  • Watch
  • Hold
  • Remember
  • Compare
  • Decide
  • Tap
  • Execute
  • Correct
  • Improve

The program supports continued participation through:

Short Training Rounds

Students receive frequent opportunities to begin again, improve, and experience progress.

Immediate Feedback

Students can see the result of their response without waiting for delayed grading.

Visible Progress

Levels, scores, and successful rounds make improvement easier to recognize.

Earned Advancement

Students must demonstrate successful performance to advance.

Personal Competition

Students can attempt to improve their own previous results.

Group Competition

Schools may organize classroom, team, grade-level, or schoolwide challenges.

Common Playing System

Because every drill uses the same grid and tempo, students spend their effort on the training instead of learning different controls.

Ritnome does not reward passive screen time. Progress must be earned through participation and performance.

10Assessment Framework

Ritnome assessment is performance-based.

Students are evaluated through what they can successfully observe, retain, recall, compare, and execute.

Primary Performance Measures

  • Accuracy
  • Correct sequence order
  • Correct target identification
  • Successful change detection
  • Completed rounds
  • Level advancement
  • Warmup reaction speed (BPM equivalent)
  • Consistency across attempts

Formative Assessment

Educators may use session results to identify:

  • Which drill is currently strongest
  • Which drill requires additional practice
  • Whether errors are caused by memory, attention, decision-making, or execution
  • Whether the student is improving over time
  • Whether the current challenge level is appropriate

Student Self-Assessment

Students may be encouraged to review:

  • Their most accurate drill
  • Their most difficult drill
  • Their personal best score
  • A mistake they corrected
  • A strategy that improved performance
  • A goal for the next session

Mastery

Mastery is demonstrated through repeated successful performance at the required level.

One correct round alone does not establish mastery. The student must show that the skill can be performed consistently.

11Suggested Curriculum Sequence

Schools may introduce the curriculum in the following order:

Phase 1: Grid Orientation

Students learn the Ritnome grid, response method, 100 BPM tempo, scoring system, and expectations for accurate play.

Phase 2: REFLEX

Students expand their attention across the grid and practice quick, accurate decisions.

Phase 3: REPLAY

Students connect complete memory retention with organized physical execution.

Phase 4: RECALL (Coming Soon)

Students begin with direct sequence memory and ordered retrieval.

Phase 5: RITMO (Coming Soon)

Students use memory, comparison, and change detection together.

Phase 6: Integrated Training

Students rotate through all four drills during the same training cycle.

Phase 7: Performance and Competition

Students apply their skills in personal challenges, classroom events, team competitions, or schoolwide Beats eSports events.

1Descripción general del currículo

El Currículo Ritnome Beats eSports es un programa de entrenamiento estructurado diseñado para desarrollar el recuerdo automático, el control de la memoria, la toma rápida de decisiones, la ejecución precisa y la participación sostenida.

El currículo está compuesto por cuatro ejercicios principales:

  • REFLEX
  • REPLAY
  • RECALL
  • RITMO

Cada ejercicio desarrolla un componente distinto del desempeño de la memoria. En conjunto, forman un sistema completo de entrenamiento que reta a los participantes a observar información, retenerla, recuperarla, compararla, tomar decisiones y ejecutar una respuesta correcta.

Los cuatro ejercicios utilizan la misma cuadrícula Ritnome, el mismo sistema de interacción y un ritmo constante de 100 pulsaciones por minuto.

Esta estructura uniforme permite que los estudiantes se concentren en el entrenamiento, en lugar de tener que aprender controles o reglas diferentes para cada actividad.

Cuatro ejercicios. Una cuadrícula. Un ritmo. Desarrollo continuo.

Estado de lanzamiento. REFLEX y REPLAY estarán disponibles en el lanzamiento como los ejercicios centrados en reacción del currículo. RECALL y RITMO se introducirán en una versión posterior como la expansión enfocada en memoria del currículo.

2Propósito del currículo

El propósito del currículo Ritnome es ayudar a los estudiantes a fortalecer su capacidad para:

  • Prestar atención a información visual.
  • Retener información en la memoria.
  • Recordar información en el orden correcto.
  • Reconocer rápidamente varios elementos de información.
  • Comparar lo que observaron anteriormente con lo que observan en el momento.
  • Convertir la información recordada en una acción precisa.
  • Mantener el desempeño bajo exigencias controladas de tiempo.
  • Mejorar mediante una práctica repetida y medible.

Ritnome aplica un principio que se utiliza comúnmente en el entrenamiento deportivo.

Los atletas repiten movimientos hasta que pueden ejecutarlos de manera automática. Ritnome utiliza la repetición estructurada para ayudar a los estudiantes a desarrollar un recuerdo más automático, preciso y confiable.

La memoria muscular hace que el movimiento sea automático. Ritnome hace que el recuerdo sea automático.

3Filosofía del currículo

Ritnome se fundamenta en cinco principios educativos.

Participación activa

Los estudiantes no pueden progresar observando de manera pasiva. Cada ronda requiere observación, memoria, toma de decisiones y respuesta.

Repetición con propósito

Los estudiantes repiten procesos esenciales de memoria, pero los patrones y los retos cambian. Esto permite practicar repetidamente sin tener que realizar exactamente la misma tarea en cada ronda.

Estructura uniforme

Cada ejercicio utiliza la misma cuadrícula y el mismo ritmo de 100 BPM. Los estudiantes pueden pasar de un ejercicio a otro sin perder tiempo aprendiendo una interfaz diferente.

Progreso medible

El desempeño se registra mediante la precisión, las rondas completadas, los niveles y las puntuaciones. Los estudiantes y educadores pueden observar el progreso a través del tiempo.

Reto progresivo

El ritmo permanece constante, pero la exigencia de memoria aumenta. Ritnome introduce automáticamente más información, secuencias más largas, patrones más complejos y mayores requisitos de dominio a medida que el estudiante mejora.

4Las cuatro áreas del currículo

EL CALENTAMIENTO: EJERCICIO FUNDACIONAL

Reacción Pura y Velocidad de Referencia

Descripción del ejercicio

Antes de cualquiera de las cuatro áreas del currículo, cada sesión de Ritnome comienza con el Calentamiento. Una sola celda se ilumina a la vez en toda la cuadrícula, y el estudiante la toca lo más rápido posible. No hay secuencia que retener, ni patrón que comparar, ni ningún requisito de memoria — el Calentamiento mide únicamente la velocidad de reacción.

Destreza principal

Velocidad de reacción y preparación para la sesión

Destrezas complementarias

  • Atención visual
  • Respuesta inmediata
  • Coordinación visomotora
  • Comodidad con el ritmo de 100 BPM
  • Confianza antes de que aumente la exigencia cognitiva

Objetivos de aprendizaje

  1. Responder a un objetivo visual lo más rápido posible.
  2. Mantener la atención durante una ronda completa de la cuadrícula.
  3. Familiarizarse con la cuadrícula, el método de respuesta y el ritmo de 100 BPM antes de introducir una tarea de memoria.
  4. Establecer una velocidad de reacción personal de referencia al inicio de cada sesión.

Un Generador de Confianza y una Herramienta de Diagnóstico

El Calentamiento cumple dos propósitos a la vez. Primero, como no requiere memoria y no puede producir un resultado desalentador, brinda a cada estudiante un logro temprano y alcanzable antes de que comiencen las áreas de memoria más exigentes. Segundo, como el Calentamiento mide el tiempo de respuesta real y lo convierte en un nivel de reacción en pulsaciones por minuto (BPM), le brinda al educador un indicador inmediato e individual de la velocidad de reacción de cada estudiante, registrado al inicio de cada sesión.

Como el nivel de reacción del Calentamiento se expresa en la misma unidad de BPM que el propio tempo de entrenamiento, la velocidad de reacción personal de un estudiante puede leerse y compararse directamente junto con el estándar fijo de 100 BPM utilizado en las cuatro áreas, sin necesidad de un instrumento de evaluación independiente.

Evidencia de preparación

El estudiante demuestra preparación cuando completa el Calentamiento para los cincuenta objetivos, con un nivel de velocidad de reacción individual, expresado en BPM, registrado para esa sesión.

Conexión educativa

  • Identificar la velocidad de respuesta actual de un estudiante antes de una tarea que requiere memoria
  • Construir un perfil personal de velocidad de reacción a lo largo del tiempo
  • Comparar la velocidad de reacción entre sesiones
  • Comenzar las áreas de memoria ya familiarizado con la cuadrícula y el ritmo

ÁREA 1: REFLEX

Decisiones rápidas y atención a toda la cuadrícula

Descripción del ejercicio

En REFLEX, varias celdas se iluminan simultáneamente. El estudiante debe identificar todas las celdas activas y tocarlas antes de que finalice la oportunidad de responder.

A diferencia de una actividad de un solo objetivo, REFLEX requiere que el estudiante observe un área más amplia, reconozca varios objetivos, tome una decisión rápida y responda sin pasar por alto ninguna parte de la cuadrícula.

Destreza principal

Toma rápida y precisa de decisiones

Destrezas complementarias

  • Reconocimiento visual inmediato.
  • Atención a varios objetivos.
  • Atención a toda la cuadrícula.
  • Rastreo visual.
  • Control de la respuesta.
  • Precisión bajo exigencias de tiempo.

Objetivos de aprendizaje

Mediante la participación en REFLEX, los estudiantes aprenderán a:

  1. Observar información en toda la cuadrícula Ritnome.
  2. Identificar varias celdas activas durante una sola presentación.
  3. Distinguir entre celdas activas e inactivas.
  4. Tomar decisiones rápidas sin tocar al azar.
  5. Responder a todos los objetivos requeridos.
  6. Mantener un balance entre rapidez y precisión.
  7. Conservar la atención a medida que aumenta la cantidad de objetivos.

Progresión

El nivel de dificultad puede aumentar mediante:

  • Una mayor cantidad de celdas iluminadas simultáneamente.
  • Mayor separación entre las celdas activas.
  • Arreglos visuales más complejos.
  • Mayores exigencias de rastreo de toda la cuadrícula.
  • Mayores requisitos de precisión.
  • Desempeño más consistente requerido antes de avanzar.

El ritmo de 100 BPM permanece igual. La dificultad aumenta porque el estudiante debe reconocer y procesar una mayor cantidad de información.

Evidencia de dominio

El estudiante demuestra dominio cuando puede identificar y seleccionar consistentemente todas las celdas requeridas, sin realizar selecciones innecesarias ni omitir objetivos.

Conexión educativa

Las destrezas desarrolladas en REFLEX pueden utilizarse cuando los estudiantes necesitan:

  • Localizar rápidamente detalles importantes.
  • Examinar una página, gráfica, diagrama o problema.
  • Identificar varios elementos correctos.
  • Tomar decisiones a partir de información visual.
  • Trabajar con precisión dentro de un periodo limitado.
  • Mantener la atención en un área visual amplia.

ÁREA 2: REPLAY

Transferencia de la memoria a la ejecución

Descripción del ejercicio

En REPLAY, el estudiante observa una secuencia completa y luego la reproduce físicamente utilizando la memoria.

Una vez finalizada la secuencia, el estudiante debe ejecutar correctamente toda la secuencia. El ejercicio enfatiza no solamente recordar la información, sino también convertirla en una acción organizada.

REPLAY reta al estudiante a retener internamente una secuencia completa y luego ejecutarla con control.

Destreza principal

Memoria y ejecución

Destrezas complementarias

  • Retención de una secuencia completa.
  • Transferencia de la memoria a la acción.
  • Ejecución ordenada.
  • Reproducción de patrones.
  • Consistencia en la respuesta.
  • Control del desempeño.

Objetivos de aprendizaje

Mediante la participación en REPLAY, los estudiantes aprenderán a:

  1. Observar una secuencia completa sin responder antes de tiempo.
  2. Retener la secuencia completa en la memoria.
  3. Organizar la información recordada antes de actuar.
  4. Ejecutar la secuencia en el orden correcto.
  5. Completar la secuencia sin perder la concentración a mitad del ejercicio.
  6. Relacionar la memoria visual con una respuesta física.
  7. Mejorar la consistencia mediante intentos repetidos.

Progresión

El nivel de dificultad puede aumentar mediante:

  • Secuencias completas más largas.
  • Más cambios de dirección en la cuadrícula.
  • Mayor distancia entre las ubicaciones de la secuencia.
  • Patrones de movimiento más complejos.
  • Mayores requisitos de precisión.
  • Una ejecución más continua requerida para demostrar dominio.

Evidencia de dominio

El estudiante demuestra dominio cuando puede retener y reproducir una secuencia completa con precisión, en el orden correcto y con una ejecución controlada.

Conexión educativa

Las destrezas desarrolladas en REPLAY pueden utilizarse cuando los estudiantes necesitan:

  • Recordar y realizar un grupo completo de instrucciones.
  • Convertir información aprendida en una acción.
  • Completar procedimientos de varios pasos.
  • Reproducir procesos demostrados.
  • Organizar sus acciones antes de comenzar una tarea.
  • Ejecutar de manera independiente una secuencia aprendida.

ÁREA 3: RECALL (PRÓXIMAMENTE)

Memoria y recuperación secuencial

Descripción del ejercicio

En RECALL, una secuencia de celdas se ilumina en la cuadrícula Ritnome. El estudiante observa la secuencia, retiene en la memoria las posiciones y el orden, y luego toca las celdas siguiendo exactamente el orden presentado.

El estudiante debe recordar:

  • Cuáles celdas aparecieron.
  • El orden en que aparecieron.

RECALL desarrolla la capacidad de recuperar información visual de manera organizada y secuencial.

Destreza principal

Memoria secuencial

Destrezas complementarias

  • Memoria de ubicación visual.
  • Recuperación ordenada de información.
  • Concentración.
  • Atención a la secuencia.
  • Precisión en la respuesta.
  • Recuerdo automático.

Objetivos de aprendizaje

Mediante la participación en RECALL, los estudiantes aprenderán a:

  1. Observar una secuencia visual desde el principio hasta el final.
  2. Retener la ubicación de cada celda presentada.
  3. Recordar el orden correcto de la secuencia.
  4. Reproducir la secuencia con precisión.
  5. Mantener la concentración a medida que las secuencias se hacen más largas.
  6. Recuperarse de los errores y mejorar en intentos posteriores.
  7. Demostrar un recuerdo cada vez más consistente.

Progresión

El nivel de dificultad puede aumentar mediante:

  • Secuencias más largas.
  • Una mayor cantidad de ubicaciones posibles en la cuadrícula.
  • Patrones de secuencia más parecidos entre sí.
  • Mayores requisitos de precisión.
  • Varias rondas exitosas requeridas para demostrar dominio.
  • Menor dependencia de demostraciones repetidas.

Evidencia de dominio

El estudiante demuestra dominio cuando puede reproducir repetidamente las secuencias del nivel requerido, manteniendo el orden correcto y un desempeño consistente.

Conexión educativa

Las destrezas desarrolladas en RECALL pueden utilizarse cuando los estudiantes necesitan:

  • Recordar los pasos de un proceso.
  • Seguir instrucciones en el orden correcto.
  • Recordar secuencias de números o letras.
  • Retener el orden de eventos.
  • Completar tareas académicas de varios pasos.
  • Recordar procedimientos sin necesidad de recibir instrucciones continuamente.

ÁREA 4: RITMO (PRÓXIMAMENTE)

Memoria, comparación y detección de cambios

Descripción del ejercicio

En RITMO, el estudiante observa un patrón original y lo retiene en la memoria. Luego, el patrón se presenta nuevamente con un cambio.

El estudiante debe comparar la segunda presentación con la primera que mantiene en la memoria e identificar exactamente qué cambió.

RITMO requiere que el estudiante cree una representación mental del patrón original, mantenga esa información, compare dos presentaciones y tome una decisión precisa.

Destreza principal

Memoria y toma rápida de decisiones

Destrezas complementarias

  • Memoria de patrones.
  • Comparación visual.
  • Detección de cambios.
  • Atención a los detalles.
  • Retención mental.
  • Toma precisa de decisiones.

Objetivos de aprendizaje

Mediante la participación en RITMO, los estudiantes aprenderán a:

  1. Observar y retener un patrón visual original.
  2. Mantener ese patrón en la memoria durante una segunda presentación.
  3. Comparar la información recordada con la información actual.
  4. Detectar el cambio específico entre dos patrones.
  5. Evitar responder a información que permaneció igual.
  6. Tomar una decisión rápida y precisa.
  7. Reconocer cambios cada vez más sutiles.

Progresión

El nivel de dificultad puede aumentar mediante:

  • Más celdas incluidas en el patrón original.
  • Arreglos de patrones más complejos.
  • Cambios más pequeños o menos evidentes.
  • Cambios ubicados en diferentes partes de la cuadrícula.
  • Mayor similitud entre el primer y segundo patrón.
  • Mayores requisitos de precisión antes de avanzar.

Evidencia de dominio

El estudiante demuestra dominio cuando puede identificar consistentemente el cambio exacto entre el patrón original y la segunda presentación.

Conexión educativa

Las destrezas desarrolladas en RITMO pueden utilizarse cuando los estudiantes necesitan:

  • Comparar dos grupos de información.
  • Detectar errores o diferencias.
  • Revisar trabajos escritos.
  • Identificar cambios en números, símbolos, diagramas o textos.
  • Distinguir información correcta de información alterada.
  • Prestar atención a detalles importantes.

Desempeño de la memoria

Los estudiantes mejorarán su capacidad para retener, recuperar y reproducir información visual.

Recuerdo automático

Los estudiantes desarrollarán mayor eficiencia al recuperar información practicada, reduciendo la duda o la demora innecesaria.

Control secuencial

Los estudiantes mejorarán su capacidad para recordar y ejecutar información en el orden correcto.

Toma rápida de decisiones

Los estudiantes tomarán decisiones con mayor rapidez, manteniendo la precisión.

Atención visual

Los estudiantes mejorarán su capacidad para observar objetivos individuales, varios objetivos, patrones completos y cambios en la cuadrícula.

Transferencia de la memoria a la acción

Los estudiantes fortalecerán la conexión entre recordar información y ejecutar una respuesta correcta.

Consistencia en el desempeño

Los estudiantes trabajarán para repetir un desempeño exitoso a través de varias rondas, en lugar de lograrlo solamente una vez.

Autoevaluación

Los estudiantes desarrollarán mayor conciencia de su precisión, progreso, fortalezas y áreas que requieren práctica adicional.

6Estructura de enseñanza

Una sesión de Ritnome puede incluir los cuatro ejercicios o concentrarse en un área específica del currículo.

La estructura recomendada para una sesión es la siguiente:

A. Preparación

El educador presenta el propósito de la sesión e identifica la destreza que se trabajará.

Ejemplos:

  • “Hoy vamos a entrenar la memoria secuencial”.
  • “Hoy vamos a trabajar la detección de cambios”.
  • “Hoy vamos a practicar decisiones rápidas sin perder precisión”.

B. Familiarización

Los estudiantes completan rondas introductorias para familiarizarse con el ejercicio y la cuadrícula.

C. Entrenamiento enfocado

Los estudiantes completan rondas estructuradas dentro del ejercicio seleccionado.

D. Reto progresivo

El sistema aumenta la dificultad cuando el estudiante demuestra el nivel de dominio requerido.

E. Revisión del desempeño

Los estudiantes revisan sus puntuaciones, precisión y niveles completados.

F. Reflexión

Los estudiantes identifican brevemente qué ayudó o afectó su desempeño.

Posibles preguntas de reflexión:

  • ¿Qué te ayudó a recordar la secuencia?
  • ¿Respondiste demasiado rápido o demasiado lento?
  • ¿Qué parte de la cuadrícula no observaste?
  • ¿Qué harás diferente en la próxima ronda?

7Ritmo estándar

Los cuatro ejercicios operan con un ritmo estándar de:

100 BPM

Los estudiantes no seleccionan niveles de velocidad para principiantes, intermedios o avanzados.

El ritmo constante crea un ambiente de entrenamiento uniforme entre:

  • Ejercicios.
  • Estudiantes.
  • Salones de clase.
  • Equipos.
  • Dispositivos.
  • Competencias escolares.

El ritmo no aumenta a medida que el estudiante avanza.

En su lugar, Ritnome aumenta la exigencia de memoria y desempeño mediante:

  • Más información.
  • Secuencias más largas.
  • Más celdas activas.
  • Patrones más complejos.
  • Comparaciones más difíciles.
  • Mayores expectativas de precisión.
  • Mayores requisitos de consistencia.

El ritmo permanece constante. El reto aumenta.

8Aumento automático del nivel de reto

Ritnome aumenta automáticamente el nivel de dificultad a medida que el estudiante demuestra dominio.

Los estudiantes no tienen que seleccionar su propia dificultad. El sistema utiliza el desempeño exitoso para determinar cuándo el estudiante está preparado para un nivel más exigente.

Según el ejercicio, la progresión automática puede incluir:

EjercicioCómo aumenta el nivel de dificultad
REFLEXMás objetivos simultáneos y mayor cobertura de la cuadrícula
REPLAYSecuencias completas más largas y patrones de ejecución más exigentes
RECALLSecuencias más largas y complejas
RITMOPatrones más complejos y cambios menos evidentes

Este modelo ayuda a evitar dos problemas comunes:

  • Que una actividad permanezca demasiado fácil y se vuelva repetitiva.
  • Que una actividad se vuelva demasiado difícil antes de que el estudiante esté preparado.

Los estudiantes progresan al demostrar desempeño, no simplemente por pasar tiempo dentro del programa.

9Sistema de participación

Ritnome está diseñado para exigir participación activa.

Cada ronda requiere que el estudiante realice una o más de las siguientes acciones:

  • Observar.
  • Retener.
  • Recordar.
  • Comparar.
  • Decidir.
  • Tocar.
  • Ejecutar.
  • Corregir.
  • Mejorar.

El programa promueve la participación continua mediante:

Rondas cortas de entrenamiento

Los estudiantes reciben oportunidades frecuentes para comenzar nuevamente, mejorar y experimentar progreso.

Retroalimentación inmediata

Los estudiantes pueden ver el resultado de su respuesta sin tener que esperar por una evaluación posterior.

Progreso visible

Los niveles, las puntuaciones y las rondas exitosas permiten reconocer fácilmente el mejoramiento.

Avance ganado

Los estudiantes deben demostrar un desempeño exitoso para avanzar.

Competencia personal

Los estudiantes pueden intentar superar sus propios resultados anteriores.

Competencia grupal

Las escuelas pueden organizar retos entre salones, equipos, grados o toda la comunidad escolar.

Sistema de juego uniforme

Como todos los ejercicios utilizan la misma cuadrícula y el mismo ritmo, los estudiantes concentran su esfuerzo en el entrenamiento y no en aprender controles diferentes.

Ritnome no premia el tiempo pasivo frente a una pantalla. El progreso debe ganarse mediante participación y desempeño.

10Marco de evaluación

La evaluación de Ritnome está basada en el desempeño.

Los estudiantes son evaluados por lo que pueden observar, retener, recordar, comparar y ejecutar correctamente.

Medidas principales de desempeño

  • Precisión.
  • Orden correcto de las secuencias.
  • Identificación correcta de objetivos.
  • Detección correcta de cambios.
  • Rondas completadas.
  • Avance de nivel.
  • Velocidad de reacción del Calentamiento (equivalente en BPM).
  • Consistencia entre intentos.

Evaluación formativa

Los educadores pueden utilizar los resultados de las sesiones para identificar:

  • Qué ejercicio representa la mayor fortaleza del estudiante.
  • Qué ejercicio requiere práctica adicional.
  • Si los errores se relacionan con memoria, atención, toma de decisiones o ejecución.
  • Si el estudiante está mejorando a través del tiempo.
  • Si el nivel actual de dificultad es adecuado.

Autoevaluación del estudiante

Se puede motivar a los estudiantes a revisar:

  • Su ejercicio con mayor precisión.
  • Su ejercicio más difícil.
  • Su mejor puntuación personal.
  • Un error que lograron corregir.
  • Una estrategia que mejoró su desempeño.
  • Una meta para la próxima sesión.

Dominio

El dominio se demuestra mediante un desempeño exitoso repetido en el nivel requerido.

Una sola ronda correcta no establece dominio. El estudiante debe demostrar que puede realizar la destreza de manera consistente.

11Secuencia sugerida del currículo

Las escuelas pueden presentar el currículo siguiendo el siguiente orden:

Fase 1: Orientación a la cuadrícula

Los estudiantes aprenden la cuadrícula Ritnome, el método de respuesta, el ritmo de 100 BPM, el sistema de puntuación y las expectativas de juego preciso.

Fase 2: REFLEX

Los estudiantes amplían su atención a toda la cuadrícula y practican la toma rápida y precisa de decisiones.

Fase 3: REPLAY

Los estudiantes conectan la retención completa de información con una ejecución física organizada.

Fase 4: RECALL (Próximamente)

Los estudiantes comienzan con memoria secuencial directa y recuperación ordenada de información.

Fase 5: RITMO (Próximamente)

Los estudiantes utilizan conjuntamente la memoria, la comparación y la detección de cambios.

Fase 6: Entrenamiento integrado

Los estudiantes rotan por los cuatro ejercicios durante un mismo ciclo de entrenamiento.

Fase 7: Desempeño y competencia

Los estudiantes aplican sus destrezas en retos personales, eventos de salón, competencias de equipo o actividades escolares de Beats eSports.

El currículo Ritnome puede implementarse en:

  • Educación física.

  • Periodos de apoyo académico.
  • Programas de tecnología.
  • Esports educativos.

  • Competencias escolares.

  • Visores PICO.
  • Computadoras de escritorio.
  • Chromebooks.
  • Pantallas interactivas (Promethean o similar).
  • Dispositivos con pantalla táctil.

Luego, los estudiantes completan sus rondas individuales de entrenamiento en sus propios Chromebooks, donde las respuestas, la precisión y el progreso de cada estudiante se registran de manera individual.

13Función del educador

El educador no tiene que dirigir cada respuesta individual. El sistema presenta los ejercicios, registra el desempeño y controla la progresión.

La función del educador es:

  • Explicar el propósito del entrenamiento.
  • Establecer expectativas de participación.
  • Promover la precisión antes que la velocidad al azar.
  • Revisar el progreso de los estudiantes.
  • Ayudar a los estudiantes a identificar estrategias útiles.
  • Organizar entrenamientos individuales o grupales.
  • Reconocer el mejoramiento y el dominio.
  • Relacionar las destrezas desarrolladas con actividades académicas y escolares.

14Identidad del currículo

Ritnome no es una colección de juegos sin relación entre sí.

Los cuatro ejercicios forman un currículo integrado:

RECALL

Recuerda el orden.

REFLEX

Encuentra todos los objetivos y decide rápidamente.

REPLAY

Retén la secuencia completa y ejecútala.

RITMO

Recuerda el patrón y detecta el cambio.

En conjunto, los ejercicios guían al estudiante a través de un ciclo completo de desempeño:

Observar. Retener. Recordar. Comparar. Decidir. Ejecutar.

El resultado es un currículo que puede crecer junto con el estudiante sin obligarlo a aprender nuevamente cómo participar.
